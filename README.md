# Projetos SQL - Business Analytics

Este repositório reúne projetos desenvolvidos com foco em prática e consolidação de prática analíticas em SQL.
Cada projeto simula casos reais de um Data Analyst com bancos de dados relacionais, desde a modelagem dos dados até o relatório final.

# PROJETOS:

1- Brazil Tech Database (Empresa Fictícia)

- Modelagem de sistema de E-commerce
- Inserção dos dados
- Queries Analíticas
- Ánalise de vendas e clientes
- Subqueries para relacionamento















📊 Objetivo: aplicar SQL desde a criação do banco até a análise, resolvendo problemas reais de business intelligence e apoiando decisões estratégicas.
